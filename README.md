# 炸金花 (Boom Golden Flower)

在线多人炸金花游戏，支持真人与AI对战。

## 运行

```bash
npm install
npm run dev      # 前端 (默认 http://localhost:5173)
npm run server   # 后端 (默认 ws://localhost:3000)
```

## 游戏规则

### 牌型大小（从大到小）
1. 豹子：三张相同点数（AAA最大，222最小）
2. 顺金：同花顺（AKQ最大，A23最小）
3. 金花：同花色不连续
4. 顺子：不同花色连续
5. 对子：两张相同点数
6. 单张：按最大牌比较

### 特殊牌型
- 235不同花色 > AAA（最大豹子）

### 下注规则
- 焖牌（不看牌下注）：下家若已看牌需跟双倍，未看牌跟同额
- 看牌后下注：下家若焖牌只需跟一半，已看牌跟同额
- 开牌：选择对手比牌，输者弃牌，费用为上家下注金额

## 功能特性

- 多人房间：支持2-6人同时游戏
- AI玩家：智能决策，分析对手行为模式
- 焖牌机制：策略性盲注玩法
- 动画效果：发牌动画、筹码飞入动画、底池数字动画
- 实时同步：WebSocket实时通信

## 技术栈

- 前端：Vue 3 + Vite
- 后端：Node.js + WebSocket (ws)
- 通信：WebSocket 双向实时通信
